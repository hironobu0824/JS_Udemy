### 関数
```js
function add(a, b) {
    return a + b;
}
```
addという変数にオブジェクトを代入しているにすぎない。
```js
let add = {}
```

```js
console.log(add);
// ƒ add(a, b) {
//     return a + b;
// }
```
```js
console.dir(add);
// ƒ add(a, b)
//     arguments: null
//     caller: null
//     length: 2
//     name: "add"
//     prototype: {constructor: ƒ}
//     [[FunctionLocation]]: main.js:1
//     [[Prototype]]: ƒ ()
//     [[Scopes]]: Scopes[1]
```
キーと値があるオブジェクトになっている。
